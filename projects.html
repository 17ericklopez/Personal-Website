My Projects

Here are some of the key projects I've worked on throughout my journey in computer science and beyond:

2-D Java Based Game

One of my favorite projects I've worked on was a 2-D game based on the John Wick movies called Juan Wick. This game was developed as part of a team project in my Software Development class. As a beginner programmer, it was a huge learning experience and gave me a deeper understanding of object-oriented programming (OOP) and game design.

Tools & Technologies Used:

  • Java: We used Java for the main game development. This included utilizing Java's built-in libraries such as Swing for graphical interfaces and event handling.
  • Object-Oriented Programming (OOP): The game structure relied heavily on OOP concepts such as classes, objects, inheritance, and polymorphism to create reusable and modular code.
  • Game Loops & Collision Detection: Implemented the core game loop to update game logic in real-time and used algorithms to detect collisions between game objects.
  • Graphics & Animation: For creating the 2D game world, we incorporated sprites, and we utilized animation techniques to bring characters and objects to life on the screen.

BEST Robotics

Another exciting project I worked on was designing and building a robot for the BEST Robotics Competition, a national competition held in the U.S. that challenges students to build robots from scratch based on certain themes and criteria. I participated in this competition in both 2016 and 2017, where I learned about hardware design, programming, and teamwork.

Tools & Technologies Used:

  • Arduino: We used Arduino microcontrollers to control the robot's movements, sensors, and actuators. Programming the Arduino was a challenge in itself but crucial for the robot’s functionality.
  • CAD Software: We used CAD software to design and plan out the robot’s structure before moving to the actual construction phase.
  • Mechanical Design & Construction: The project involved building the robot’s frame, setting up motors, and attaching sensors based on the challenge’s theme.
  • Teamwork & Problem-Solving: This project helped improve my communication and problem-solving skills as we worked in a team to meet deadlines and adapt to unforeseen challenges.

GuitaR-AI (Currently)

Currently, I am working on an exciting project called GuitaR-AI, which aims to revolutionize how people learn guitar. The idea is to create a program that can isolate the guitar sound from a music track, separate it from other instruments, and then train an AI model to recognize guitar chords and notes. This way, users can easily learn how to play along with their favorite songs.

Tools & Technologies Used:

  • Python: Python is being used for data processing, machine learning, and audio manipulation. Libraries like librosa and pydub help process and separate the guitar audio from other tracks.
  • AI & Deep Learning: The AI model will be trained using deep learning techniques, particularly convolutional neural networks (CNNs), to recognize guitar chords from audio data.
  • Music Theory: I’m also integrating some basic music theory into the program to ensure the AI can learn and play in a musically correct way.
  • TensorFlow/Keras: These libraries will be used to build and train the machine learning models to analyze the guitar sounds and output the correct chords.
